# Product Requirements: xueba

This project is a single-file HTML5 Canvas learning game deployed through GitHub Pages.

Canonical detailed product/game design lives in:

- `.vibemgmt/GDD_ADDENDUM_2026-06-13_strategy-depth.md`
- `.vibemgmt/RESEARCH_2026-06-14_kid-friendly-enrichment.md`

## Goals

- Keep the game kid-friendly and original.
- Preserve the single-file deployment model unless explicitly changed.
- Support campaign progression, collection, learning review, and lightweight replayability.
- Maintain GitHub Pages compatibility.

## Acceptance

- `index.html` remains directly runnable in a browser.
- Save import/export stays backward compatible.
- Mobile landscape flow remains usable.
- Reviews and QA evidence stay under `.vibemgmt/reviews/`.
