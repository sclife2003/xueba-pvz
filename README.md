# Xueba Campus Adventure

Single-file HTML5 Canvas tower-defense game for GitHub Pages.

## Play

Open the live page:

https://sclife2003.github.io/xueba-pvz/

Or run locally by opening `index.html` in a browser.

## Project Layout

- `index.html` - complete game runtime.
- `assets/` - static assets used by the page.
- `.vibemgmt/` - project memory, design notes, reviews, tickets, fixes, handoffs, and decisions.
- `MEMORY.md` - pointer to the canonical project memory at `.vibemgmt/MEMORY.md`.

## Notes

This project is an original school-themed lane defense game. It does not copy Plants vs. Zombies assets, characters, or names.
